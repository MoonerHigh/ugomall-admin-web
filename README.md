# 优购商城后台管理系统

[https://github.com/MoonerHigh/ugo-mall-ui](https://github.com/MoonerHigh/ugo-mall-ui)

## 安装

NodeJs版本号为：12.x或者14.x

```
# 切换到项目根目录

# 安装插件npm install

# 启动项目npm run serve
```

启动完成后，访问http://localhost:8081
